# CSS 頭文字 D 甩尾特效

從速度競賽的標準來看，刻意甩尾過彎只不過是一種花招，這不僅無法讓速度加快還會損及車輛通過彎道的時間，但若從安全的角度來看，甩尾就等於車尾打滑，若懂得如何將甩出去的車尾給救回來，對行車安全反而能帶來不小的幫助。

## 使用方式
- 把整個專案複製一份到你的電腦裡，這裡指的「內容」不是只有檔案，而是指所有整個專案的歷史紀錄、分支、標籤等內容都會複製一份下來。
```sh
$ git clone
```
